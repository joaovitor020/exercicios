A tag <ol> significa lista ordenada.
Dentro dessa lista ordenada, podem ter até 5 tipos de caracteres no comando type para colocar na lista (1, A, a, I, i).
O start dentro do type significa a partir de que caracter(seja número ou letra) a lista deve iniciar.

A tag <ul> significa lista não ordenada.
Dentro dessa lista não ordenada, podem ter vários tipos de caracteres no comando type para colocar na lista(disc, circle, square).
